OrgTreeChart
================================
    该项目主要是使用jquery实现组织架构的树形显示。

使用说明
--------------------------------    
    把一定格式的json组织结构转化成树形组织架构，并在浏览器显示。
    代码中附插件和demo。
    该插件参考[jOrgChart](https://github.com/wesnolte/jOrgChart)<br/>, 是在jOrgChart的基础上扩展了支持了JSON源数据
